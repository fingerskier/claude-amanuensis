# claude-amanuensis
Claude as a writing assistant

## Review
1. Developmental (structure/arc)
2. Line (sentence craft)
3. Copy (grammar/consistency)
4. Proofread (types)

> ...these need to be separate invocations, with clean context, to yield best results

## Allies and Adversaries
* Invoke `codex` and/or `gemini` to get high-value second opinions
* Instantiate multiple agents and collate their results

## Anchoring
* Glean examples of the author's work
* Agents' goal should be [partly] to preserve the idiosyncratic word choices, rhythm and deliberate errata
* Score any output on its _voice fidelity_
* Maintain a style-sheet for each author and/or output form (fiction, technical, long, casual)

## Diff Always
* Generate a diff or numbered list of proposed edits first
* Rewrite only from such a list

## Scribing
* Expand the author's dictation or sketch into prose
* Compress the prose in accordance with voice and style
